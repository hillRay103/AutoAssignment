# AutoAssignment
AutoAssignment is a VS Code extension that instantly generates formatted assignment documents from your code and output. No more manual copy-pasting — just fill in your details, load your files, and export to Word or PDF in seconds.


It started as my own frustration — spending more time formatting assignments than actually solving them. I looked around and realized thousands of CS students across Pakistan (and beyond) face the exact same thing. Every submission, same repetitive copy-paste, same manual formatting, same wasted time.
That's the problem I care about: the small inefficiencies that seem minor alone but multiply across millions of students and hours. Technology should eliminate that friction entirely — not just reduce it, but make it invisible.
If a task is repeated, it can be automated. If it's automated well, it scales to everyone.
